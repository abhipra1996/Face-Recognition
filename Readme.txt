Modules ,Software and Hardware Required:-
1) Python 2.7
2) Open CV
3) Sklearn,Scikit,Numpy Libraries
4) Integrated Webcam
5) Anaconda

For 2-class Classifier use Training_XMatrix256_AR_8,Training_YMatrix256_AR_8,Name_labels_256_AR_8.
For 45-class Classifier use Training_XMatrix256_64,Training_YMatrix256_64,Name_labels_256_64.

Steps To run:-
1) Open facedetect.py in Spyder.
2) Change the path of haarcascadefrontalface module to the location where it's availiable on your system.
3) Choose the classifier you want to run and modify it in Train_X,Train_Y and Naming_Labels respectively for loading the training data Otherwise for new data open       database.py and change the path to new database and run it to obtaib the new Training Matrix and use it in facedetect.py .
4)  Click on the run button.
